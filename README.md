# ERICA_GPA-Calculator

#### 한양대학교 ERICA 소프트웨어 학부생들을 위한 학점 계산기

소프트웨어 학부 학사에서 학기별로 수강을 권장하는 전공과 기초필수 과목들의 과목명과 학점 정보를 데이터베이스에 저장했다.

계산하고자 하는 학년과 학기를 선택하면, 해당 학기의 수강 권장 과목들을 버튼만 눌러서 간편하게 선택할 수 있다.
